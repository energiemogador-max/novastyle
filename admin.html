✅ Injection automatique
En cliquant "Enregistrer", les pixels sont sauvegardés dans Firebase et dans le fichier /ads_config.json sur GitHub. Le script /assets/ads-loader.js inclus sur toutes les pages lit ce fichier et injecte automatiquement les trackers actifs — aucune action supplémentaire requise.
